## ✨ Features

- 🎨 **Fully customizable** — name, date, time, location, accent color
- ⏱ **Live countdown timer** — ticks down to your event in real time
- 🌗 **Dark, Light, or Toggle mode** — set it in the config
- 📍 **Google Maps link** — drop in your maps URL
- 🔗 **RSVP button** — link to a form, email, or any URL
- 📱 **Fully responsive** — looks great on mobile and desktop
- ⚡ **Zero dependencies** — pure HTML, CSS, and JS

## 🚀 Quick Start

1. Clone or download the repo
2. Open `index.html` and edit the `CONFIG` block at the top
3. Deploy to GitHub Pages and share your link!
